## 核心技术栈

- **Vue 3** + **TypeScript** + **Vite** - 前端框架
- **Vue Router** - 页面路由
- **GSAP** - 动画效果（滚动触发、时间轴动画）
- **date-fns** - 日期处理
- **github-markdown-css** - 美观的文字展示
- **Pinia** - 状态管理（如需本地存储模式）

## 功能模块说明

### 1. Timeline 时间轴页面 (`src/views/Home.vue`)

- 垂直时间轴布局，支持滚动触发动画
- 时间节点分为"单点"和"区间"两种类型
- 每个节点可展开查看关联的框架内容
- 框架支持富文本和图片展示
- 筛选功能：按人物、按时间范围

### 2. 人物管理页面 (`src/views/People.vue`)

- 人物卡片展示（头像、姓名、星座徽章）
- 添加/编辑人物表单
- 星座根据生日自动计算
- 关联的时间节点展示

### 3. 动态效果规划

- **入场动画**：滚动到可视区域时，时间轴节点从侧边滑入
- **连线动画**：时间轴线随滚动逐渐绘制
- **悬浮效果**：节点卡片悬浮时轻微放大 + 阴影增强
- **日期高亮**：今日日期自动高亮显示

### 4. 星座计算逻辑 (`src/composables/useZodiac.ts`)

- 根据生日日期计算太阳星座
- 边界日期正确处理（如 3月21日是白羊/双鱼分界）
- 返回星座名称和图标

## 部署配置

### GitHub Actions 工作流 (`.github/workflows/deploy.yml`)

- 推送代码到 main 分支时自动构建
- 构建产物部署到 gh-pages 分支
- GitHub Pages 从 gh-pages 分支提供服务

### Vite 配置调整

- 设置 `base: '/timeline/'` 适配 GitHub Pages 路径
- 配置静态资源处理

## 交互设计要点

1. **时间节点选择器**：使用原生 date input 或轻量级日期选择组件
2. **框架编辑**：使用 contenteditable 或轻量级编辑器
3. **图片上传**：由于静态托管限制，提供两种方式：
  - 放入 `public/uploads` 目录后重新部署
  - 或转 Base64 存入 JSON（适合小图）
